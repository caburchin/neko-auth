# Neko-Auth

Supabase and Next.js App Router Sample.

## Requirement

- Node.js v20
- Supabase Account

## Dependencies

- Next.js (App Router)
- supabase-js
- pico.css
- Jest
- Storybook
- Radix

## Storybook

```
npm run storybook
```

## Test

```
cp .env.example .env.test.local
# Edit .env.test.local
npm run test
```

## Lint

```
npm run lint
```

## Run locally (for dev use)

```
cp .env.example .env.local
# Edit .env.local
npm run dev
```

## Deploy

Push to main branch and it will be deployed by Vercel.
